### なぜMyClawがすべてを変えるのか

ほとんどのAIツールはチャットウィンドウを提供します。MyClawは**サーバー**を提供します。

すべてのMyClawインスタンスは[OpenClaw](https://github.com/openclaw/openclaw) — オープンソースのAIエージェントランタイム — を専用インフラストラクチャ上で実行します。あなたのエージェントには：

- 🖥️ **完全なLinuxサーバー** — サンドボックスでもコンテナでもない、本物のマシン
- 🌐 **制限のないインターネット** — ブラウズ、スクレイプ、API呼び出し、メッセージ送信
- ⏰ **Cronとバックグラウンドタスク** — あなたが寝ている間もエージェントが働く
- 📁 **永続的なファイルシステム** — ワークスペース、メモリ、設定、すべてあなたのもの
- 🧩 **スキルエコシステム** — [ClawHub](https://myclaw.ai/skills)から一つのコマンドで新しい能力をインストール
- 🔌 **チャンネル連携** — Telegram、Discord、Slack、WhatsAppなど
- 🔒 **あなたのデータ、あなたのサーバー** — 共有インフラなし、データマイニングなし

これがAuto-Dreamが存在する理由です。あなたのAIエージェントに永続的なサーバー、永続的なファイル、そしてあなたとの永続的な関係がある時 — **記憶はあったら便利なものではありません。知性の基盤なのです。**

## 問題

すべてのAIエージェントは忘れます。セッションが終了すれば、コンテキストは消えます。ファイルが溜まっていきます。2週間前のあの決定は何だったか？前回どのワークフローがうまくいったか？あなたのエージェントは健忘症です — 機能はしていますが、眠りにつく瞬間にすべてを忘れてしまいます。

**Auto-Dreamがこれを解決します。** 人間の脳が睡眠中に記憶を統合するように、Auto-Dreamは定期的な「ドリームサイクル」を実行し、エージェントの知識をスキャン、抽出、整理、スコアリング、リンク、プルーニングします — 自動的に、安全に、インテリジェントに。

### MyClawユーザーにとってなぜ重要なのか

あなたのMyClawエージェントは24時間365日稼働しています。プロジェクトを管理し、決定を追跡し、好みを記憶し、ワークフローを管理します。数週間、数ヶ月にわたり、数千の日次ログエントリが蓄積されます。Auto-Dreamがなければ、その知識は生のファイルに眠ったまま — 未整理、未評価、未接続。エージェントはデータを持っていますが、それについて*考える*ことができません。

Auto-Dreamはあなたのエージェントをコマンドを実行するツールから、**コンテキストを理解し、履歴から学び、あなたが見逃した点と点を結ぶ**パートナーへと変身させます。

## 機能

### 🧠 認知ドリームサイクル

cron経由で自動実行（デフォルト：毎日午前4時）。3つのフェーズ：

1. **収集** — 未統合の日次ログ（過去7日間）をスキャン、優先度マーカー（`⚠️ PERMANENT`、`🔥 HIGH`、`📌 PIN`、`<!-- important -->`）を検出、決定 / 人物 / 事実 / プロジェクト / 教訓 / 手続き / 未解決スレッドを抽出

2. **統合** — 各インサイトを適切な記憶層にルーティング、意味的重複排除を実行、一意のID（`mem_NNN`）を割り当て、接続されたエントリ間の関係リンクを作成

3. **評価** — `base_weight × recency × reference_boost / 8.0` で重要度をスコアリング、忘却曲線を適用（>90日 + 低重要度 → アーカイブ、決して削除しない）、5指標ヘルススコアを計算、1〜3の非自明な観察を生成、ドリームレポートを作成、通知を送信

### 📊 重要度スコアリング

すべてのエントリがドリームサイクルごとにスコアを受け取ります：

```
importance = (base_weight × recency_factor × reference_boost) / 8.0
```

- **新しさ**: `max(0.1, 1.0 - days/180)` — 6ヶ月にわたる段階的な減衰
- **参照数**: `log₂(count + 1)` — 頻繁に参照されるエントリへの対数ブースト
- **マーカー**: `🔥 HIGH` は基本重みを2倍に; `⚠️ PERMANENT` は常にスコア1.0

### 📉 インテリジェントな忘却

記憶は削除されません — 優雅にアーカイブされます：
- エントリが >90日間未参照 かつ 重要度 < 0.3 である必要があります
- `memory/archive.md` に1行の要約に圧縮されます
- 関係追跡のためにオリジナルIDが保持されます
- `⚠️ PERMANENT` と `📌 PIN` のエントリは免除されます

### 🕸️ 知識グラフ

エントリは意味的関係によってリンクされます。到達可能性メトリクスはグラフの接続性を測定します：
- すべてのエントリ関係にわたるUnion-Findアルゴリズム
- 孤立した知識クラスターを検出
- 一貫性を向上させるための相互参照を提案

### 🩺 ヘルススコア（5指標）

```
health = (freshness×0.25 + coverage×0.25 + coherence×0.2 + efficiency×0.15 + reachability×0.15) × 100
```

| 指標 | 測定内容 |
|------|---------|
| **鮮度** | 過去30日間に参照されたエントリの% |
| **カバレッジ** | 過去14日間に更新された知識カテゴリの% |
| **一貫性** | 少なくとも1つの関係リンクを持つエントリの% |
| **効率性** | MEMORY.mdの簡潔さ（行数に反比例） |
| **到達可能性** | 記憶グラフの接続性の良さ |

### 🔔 プッシュ通知

ドリーム結果はチャットに自動配信されます：

| レベル | 受け取る内容 |
|--------|-------------|
| `silent` | なし — dream-log.mdにのみ記録 |
| `summary` | `🌀 ヘルス: 82/100 \| +5 新規, ~3 更新, -1 アーカイブ \| 💡 トップインサイト` |
| `full` | すべてのセクションを含む完全なドリームレポート |

### 📊 インタラクティブダッシュボード

依存関係のないHTMLダッシュボード：
- アニメーション付きヘルスゲージ（0〜100）
- カラーコード付き5つのメトリクスカード
- 記憶分布ドーナツチャート
- 重要度ヒストグラム
- ヘルストレンド折れ線グラフ（過去30サイクル）
- 力指向知識グラフ可視化
- 最近の変更、インサイト、提案、古いエントリ

生成方法: *「記憶ダッシュボードを表示」*

### 🔄 クロスインスタンス移行

MyClawインスタンス間で記憶を移動：

```
"Export memory bundle"    →  memory/export-2026-03-28.json
"Import memory bundle"    →  既存とマージ（新しい方が優先）
"Export only procedures"  →  選択的な層エクスポート
```

完全なメタデータ、競合解決、インポート前バックアップを備えたポータブルJSONバンドル形式。複数のMyClawインスタンスを管理したり、新しいサーバーにアップグレードするユーザーに最適です。

### 🔮 ドリームインサイト

各サイクル後に1〜3の非自明な観察：
- **パターン接続** — *「プロジェクトXの戦略はプロジェクトYで成功したものを反映している」*
- **時間的トレンド** — *「戦略的決定が月曜日に集中 — 週次計画を検出」*
- **ギャップ検出** — *「過去4プロジェクトで教訓の記録なし — 振り返りが延滞」*
- **トレンドアラート** — *「ヘルスが3サイクル低下中: 85→79→72 — 古いエントリが蓄積」*
- **密度分析** — *「mem_042は8エントリから参照されているが、外向きリンクがない」*

## 安全性

| ルール | 理由 |
|--------|------|

| `⚠️ PERMANENT` を絶対に削除しない | ユーザー保護は絶対 |

## 内部の仕組み

Auto-DreamはOpenClaw のネイティブプリミティブを活用します — すべてのMyClawインスタンスを動かすのと同じインフラストラクチャです：

| プリミティブ | 役割 |
|-------------|------|
| **Cron** | 設定可能な間隔でドリームサイクルをスケジュール |
| **分離セッション** | メインチャット履歴を汚染せずに統合を実行 |
| **ファイルシステム** | 5つの全層にわたって記憶ファイルを読み書き |
| **LCM** | 長い会話のためのワーキングメモリ圧縮を提供 |

外部依存関係なし。APIキーなし。データベースなし。ファイルとインテリジェンスだけ。
